# Prosthetic Outreach Volunteer Coordination

## Overview
This repository coordinates volunteer tasks across our prosthetic outreach initiative teams in **Chicago, Boston, and Atlanta**.

## Quick Start
1. Browse the [Tasks](#tasks) section to find assignments
2. Claim a task by commenting on the issue
3. Update progress using issue labels:
   - 🔴 `urgent` — High priority
   - 🟡 `in-progress` — Currently working on
   - 🟢 `completed` — Finished
   - 🔵 `discussion` — Needs team input
